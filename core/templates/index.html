Port E-Commerce

Site E-Commerce desenvolvido com Python-Django, pra fins de Portifólio.

neste projeto usei como stack principal: Python-Django|Javascript-Jquery/Ajax.

Nele você consegue criar um usuário, editar todos os dados deste usuário, criar um carrinho de compras, fazer checkout e até simular um pagamento usando o Paypal.

O intuito foi desenvolver o backend de um E-commerce usual(por isto não foquei no Frontend), neste projeto consigo expor grande parte do conhecimento que tenho com no Framework Django.
